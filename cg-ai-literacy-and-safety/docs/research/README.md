## Project Summary

### Selected Game: Bias Bounty Lite

**Tagline:** *Find the hidden unfairnessâ€”before it finds you.*

**Core Innovation:** Three intuitive patterns replace technical bias terminology:
- ðŸ”´ **BAD START** - Learned from biased history
- ðŸŸ¡ **WRONG MEASURING** - Measures success poorly  
- ðŸŸ¢ **SNEAKY SHORTCUTS** - Finds hidden ways to discriminate

**Key Design Decisions:**
1. **Collaborative, not competitive** - Group investigates together
2. **No hidden information** - All cards visible, no memory burden
3. **Warm-up first** - Non-AI examples teach pattern recognition
4. **Human impact focus** - Every problem shows WHO was affected
5. **Standards-backed** - Aligns with UNESCO, OECD, NIST, EU frameworks

### Game Specs
| Attribute | Value |
|-----------|-------|
| Players | 3-6 (best: 4-5) |
| Time | 25-35 minutes |
| Age | 12+ |
| Components | 40 cards + rulebook |
| Complexity | Light-medium |

### What's in the Final Design Document

The complete production specification includes:
- **8 System cards** (Hiring, Loan, Health, School, Crime, Insurance, Content, Voice)
- **24 Problem cards** (3 per system, real-world examples)
- **3 Type reference cards** (BAD START, WRONG MEASURING, SNEAKY SHORTCUTS)
- **3 Warm-up cards** (Non-AI examples for teaching patterns)
- Complete rules and facilitation guide
- Accessibility features
- Production specifications
- Expansion roadmap
